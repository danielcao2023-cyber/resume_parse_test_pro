# AI Resume Parser（AI简历自动解析系统）

## 🚀 项目简介

本项目是一个基于大模型（Gemini）的智能简历解析系统，支持对 PDF / Word / 图片格式的简历进行自动识别与结构化提取，并实现批量处理与数据入库。

该系统已在真实招聘场景中设计与验证，适用于高校招聘、人力资源管理及人才数据分析。

---

## 🎯 核心功能

* ✅ 支持多格式简历解析（PDF / DOCX / 图片）
* ✅ 自动提取关键信息（姓名 / 电话 / 学历 / 工作经历等）
* ✅ 支持**一份文件多简历自动拆分**
* ✅ 自动去重（基于电话 / 邮箱）
* ✅ 自动分类（已处理 / 失败 / 重复）
* ✅ 批量处理（支持多线程）
* ✅ 自动写入 Excel（结构化数据）
* ✅ AI辅助人才评估与推荐（博士/年龄/专业匹配）

---

## 🧠 AI能力亮点

本项目不仅完成信息提取，还实现了：

* 🎯 学历判断（博士识别）
* 🎯 年龄评估（是否符合招聘政策）
* 🎯 专业匹配（自动匹配学院）
* 🎯 招聘建议（A/B/C/D分级）
* 🎯 AI评分与推荐逻辑

👉 可直接作为招聘决策辅助工具

## ⚙️ 使用方法

### 1️⃣ 安装依赖

```bash
pip install -r requirements.txt
```

### 2️⃣ 配置 API Key

在代码中设置：

```python
GEMINI_API_KEY = "YOUR_API_KEY"
```

（请使用你自己的 API Key）

---

### 3️⃣ 运行程序

```bash
python resume_parse_test_pro.py
```

---

### 4️⃣ 使用方式

将简历文件拖入：

```
resume_input/
```

系统会自动：

* 解析简历
* 写入 Excel
* 分类文件（processed / failed / duplicated）
