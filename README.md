# AI Resume Parser (简历自动解析工具)

## 项目简介
本项目是一个基于 AI 的简历解析工具，支持将 PDF / Word / 图片格式的简历自动提取为结构化数据，并写入表格系统（如 Google Sheets）。

适用于：
- 招聘会现场快速录入简历
- HR 批量处理候选人信息
- 自动化人才数据管理

---

## 功能特点
- 支持多格式简历（PDF / DOCX / 图片）
- 自动提取姓名、电话、学历、工作经验等关键信息
- 可扩展接入 Google Sheets / Excel
- 支持批量处理
